# dapp

Our Donate Crypto web3 frontend. Project from Web3 Week: https://www.luiztools.com.br/w3w

## How to Run

1. git clone dapp
2. npm install
3. If you have your contract deployed, change his address at services/Web3Service.js and your ABI.json
4. npm run dev
